Steps to run the application:
1. Navigate to project folder
2. Run 'composer update' in terminal or command prompt
3. If using XAMPP, create database called 'bosscuts' and use bosscuts.sql in the root directory to import the tables and data.
    Otherwise, run migrations. However the databse will be empty and the details like services and prices will not be present.
